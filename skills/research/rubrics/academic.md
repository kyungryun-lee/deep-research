# 학술 리서치 평가 기준 (Academic Rubric)

학술 논문, 연구 동향, 이론적 기반 조사에 특화된 평가 기준입니다.
Literature와 References의 가중치가 높고, peer-reviewed 소스를 중시합니다.

## 가중치 (Evaluator 자동 추출용)

```json
{"weights":{"scope":0.10,"literature":0.25,"analysis":0.20,"recency":0.15,"actionability":0.10,"organization":0.05,"references":0.15},"sea_threshold":85}
```

## 차원별 기준

### 1. Scope (범위) — 가중치 10%
범용 rubric과 동일하되, **관련 연구 분야의 교차 영역** 포함 여부를 추가 평가합니다.

### 2. Literature (문헌 품질) — 가중치 25% (↑)
| 점수 | 기준 |
|------|------|
| 90-100 | S등급(peer-reviewed) 50%+, 핵심 컨퍼런스(NeurIPS/ICML/ICLR/ACL/AAAI) 논문 3편+, 서베이 논문 1편+ |
| 70-89 | S등급 30%+, 핵심 컨퍼런스 1-2편, arXiv 프리프린트 다수 |
| 50-69 | arXiv 위주, peer-reviewed 부족 |
| 30-49 | 블로그 위주, 학술 소스 극소 |
| 0-29 | 학술 소스 없음 |

### 3. Analysis (분석 깊이) — 가중치 20%
범용 rubric과 동일하되, **연구 방법론 비교**, **한계점 식별**, **오픈 문제 도출** 여부를 추가 평가합니다.

### 4. Recency (최신성) — 가중치 15%
범용 rubric과 동일하되, **기초 논문(foundational)은 오래되어도 허용**합니다. 최신 논문이 인용하는 기초 논문은 감점하지 않습니다.

### 5. Actionability (실행가능성) — 가중치 10% (↓)
학술 리서치에서는 이론적 이해가 우선입니다. 구현 가능성보다 **개념적 명확성**을 평가합니다.

### 6. Organization (구조) — 가중치 5%
범용 rubric과 동일.

### 7. References (참조 정확성) — 가중치 15% (↑)
| 점수 | 기준 |
|------|------|
| 90-100 | 모든 논문 URL/DOI 유효, 저자/연도/venue 정확, 할루시네이션 0 |
| 70-89 | URL 대부분 유효, 1건 부정확 |
| 50-69 | 일부 논문 정보 부정확 |
| 30-49 | 존재하지 않는 논문 인용 |
| 0-29 | 대부분 할루시네이션 |

## SEA 충족률 기준
- 학술 rubric에서는 **85% 이상**이 충분 기준 (더 엄격)

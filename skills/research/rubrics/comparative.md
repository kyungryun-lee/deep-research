# 비교 분석 평가 기준 (Comparative)

다중 대안 비교, 의사결정 지원 리서치를 위한 평가 기준입니다.
Coverage(범위)와 Depth(분석깊이), Objectivity(객관성)의 가중치가 높습니다.

## 가중치 (Evaluator 자동 추출용)

## 맥락 확장 특이사항

### Objectivity (객관성) — 8%
| 점수 | 기준 |
|------|------|
| 90-100 | 모든 대안에 동등한 깊이, 장단점 균형, 편향 없음 |
| 70-89 | 대체로 균형적, 1개 대안에 약간 편중 |
| 50-69 | 특정 대안에 명확히 편향, 불리한 데이터 누락 |
| 0-49 | 한쪽만 긍정적, 다른 대안 무시 |

### Depth (분석깊이) — 10%
| 점수 | 기준 |
|------|------|
| 90-100 | Trade-off 매트릭스, 상황별 추천, 비용/성능 정량 비교 |
| 70-89 | 비교 분석 있으나 일부 정성적 |
| 0-69 | 단순 나열, 비교 기준 불명확 |

### Coverage (범위) — 12%
비교 분석에서는 모든 주요 대안을 빠짐없이 다루는 것이 핵심.

### Structure & Coherence — 5%
비교 분석은 구조(비교표, 매트릭스)가 가독성에 직접 영향.

## SEA 충족률 기준
- 75% 이상: 충분

## 캘리브레이션 앵커 (Evaluator 참고용)

### 점수 90 예시 (High Quality)
- 모든 대안에 동일한 기준(성능/비용/사용성/제약)으로 정량 비교
- Trade-off 매트릭스 + 상황별 추천 ("X 상황이면 A, Y 상황이면 B")
- 편향 없음 — 각 대안의 강점과 약점을 균형있게 기술
- 벤치마크 데이터 포함, 조건/범위 명시

### 점수 70 예시 (Acceptable)
- 주요 대안 비교 있으나 일부 정성적
- 비교표 있으나 일부 기준 누락
- 대체로 균형적이나 1개 대안에 약간 편중

### 점수 50 예시 (Needs Improvement)
- 대안 일부 누락, 비교 기준 불명확
- 정성적 비교만, 정량 데이터 없음
- 특정 대안에 편향

### 점수 30 예시 (Poor)
- 한 대안만 상세 기술, 나머지 무시
- 비교 없이 단순 나열
- 데이터 없는 주관적 추천

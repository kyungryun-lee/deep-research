# 범용 리서치 평가 기준 (Default Rubric)

모든 유형의 리서치에 적용되는 균형 잡힌 평가 기준입니다.

## 가중치 (Evaluator 자동 추출용)

```json
{"weights":{"scope":0.15,"literature":0.20,"analysis":0.20,"recency":0.15,"actionability":0.15,"organization":0.05,"references":0.10},"sea_threshold":80}
```

## 차원별 기준

### 1. Scope (범위) — 가중치 15%
| 점수 | 기준 |
|------|------|
| 90-100 | 질문의 모든 하위 영역을 포괄하며, 관련 배경까지 포함 |
| 70-89 | 주요 영역 대부분 포함, 1-2개 부차적 영역 누락 |
| 50-69 | 핵심 영역은 포함하나, 중요한 관점 일부 누락 |
| 30-49 | 질문의 일부만 다룸, 주요 영역 누락 |
| 0-29 | 질문과 관련성이 낮거나 핵심 누락 |

### 2. Literature (문헌 품질) — 가중치 20%
| 점수 | 기준 |
|------|------|
| 90-100 | S/A 등급 80%+, 다양한 소스 유형, 충분한 수량 |
| 70-89 | S/A 등급 60-79%, 주요 소스 유형 포함 |
| 50-69 | S/A 등급 40-59%, 소스 다양성 부족 |
| 30-49 | S/A 등급 40% 미만, B/C 의존 |
| 0-29 | C등급 위주이거나 소스 수 극히 부족 |

### 3. Analysis (분석 깊이) — 가중치 20%
| 점수 | 기준 |
|------|------|
| 90-100 | 교차 분석, 모순 식별, 패턴 도출, 통합적 해석 |
| 70-89 | 주요 발견 간 연결, 일부 교차 분석 |
| 50-69 | 소스별 요약은 있으나 통합 분석 부족 |
| 30-49 | 단순 나열 위주, 분석적 해석 거의 없음 |
| 0-29 | 복사-붙여넣기 수준, 분석 없음 |

### 4. Recency (최신성) — 가중치 15%
| 점수 | 기준 |
|------|------|
| 90-100 | 요구 기간 내 소스 90%+, 최신 동향 반영 |
| 70-89 | 요구 기간 내 70-89%, 핵심 최신 소스 포함 |
| 50-69 | 요구 기간 내 50-69%, 일부 오래된 소스 |
| 30-49 | 오래된 소스 위주, 최신 정보 부족 |
| 0-29 | 대부분 오래된 정보, 현재 상황 미반영 |

### 5. Actionability (실행가능성) — 가중치 15%
| 점수 | 기준 |
|------|------|
| 90-100 | 코드 예시, API 명세, 구체적 구현 가이드 포함 |
| 70-89 | 구현 방향은 명확, 일부 구체적 가이드 |
| 50-69 | 개념적 방향 제시, 구체적 구현 정보 부족 |
| 30-49 | 추상적 설명 위주, 실행에 활용 어려움 |
| 0-29 | 순수 이론, 실행 정보 없음 |

### 6. Organization (구조) — 가중치 5%
| 점수 | 기준 |
|------|------|
| 90-100 | 논리적 흐름, 중복 없음, 명확한 구분 |
| 70-89 | 대체로 구조적, 약간의 중복 |
| 50-69 | 구조 있으나 흐름이 부자연스러움 |
| 30-49 | 비구조적, 내용 반복 |
| 0-29 | 혼란스러운 구조 |

### 7. References (참조 정확성) — 가중치 10%
| 점수 | 기준 |
|------|------|
| 90-100 | 모든 URL 유효, 인용과 원문 일치, 할루시네이션 0 |
| 70-89 | URL 대부분 유효, 1-2건 미확인 |
| 50-69 | 일부 URL 접근 불가, 인용 부정확 |
| 30-49 | 다수 URL 무효, 할루시네이션 의심 |
| 0-29 | 대부분 검증 불가, 할루시네이션 다수 |

## SEA 충족률 기준

- 80% 이상: 충분 (is_sufficient = true)
- 60-79%: 부분 충족 (보완 검색 권장)
- 60% 미만: 불충분 (보완 검색 필수)

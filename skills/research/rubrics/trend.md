# 트렌드/동향 평가 기준 (Trend Rubric)

최신 동향, 시장 흐름, 미래 전망 조사에 특화된 평가 기준입니다.
Recency와 Scope의 가중치가 높고, 다양한 시각을 중시합니다.

## 가중치 (Evaluator 자동 추출용)

```json
{"weights":{"scope":0.20,"literature":0.15,"analysis":0.15,"recency":0.25,"actionability":0.10,"organization":0.05,"references":0.10},"sea_threshold":75}
```

## 차원별 기준

### 1. Scope (범위) — 가중치 20% (↑)
| 점수 | 기준 |
|------|------|
| 90-100 | 기술+시장+커뮤니티+기업 동향 모두 포함, 글로벌+로컬 시각 |
| 70-89 | 주요 관점 3개+ 포함 |
| 50-69 | 1-2개 관점만 |
| 30-49 | 단일 시각 |
| 0-29 | 편향된 관점 |

### 2. Literature (문헌 품질) — 가중치 15%
| 점수 | 기준 |
|------|------|
| 90-100 | 산업 보고서(Gartner/Anthropic/GitHub) + 컨퍼런스 키노트 + 데이터 기반 분석 |
| 70-89 | 산업 보고서 1-2개 + 다수 엔지니어링 블로그 |
| 50-69 | 블로그 위주, 공식 보고서 부족 |
| 30-49 | 비검증 소스 위주 |
| 0-29 | 소스 부족 |

### 3. Analysis (분석 깊이) — 가중치 15%
범용 rubric과 동일하되, **시간축 분석** (과거→현재→미래), **비교 관점** (이전 vs 현재 패러다임)을 추가 평가합니다.

### 4. Recency (최신성) — 가중치 25% (↑↑)
| 점수 | 기준 |
|------|------|
| 90-100 | 최근 6개월 내 소스 70%+, 최근 1주-1개월 소스 포함 |
| 70-89 | 최근 1년 내 70%+, 최근 3개월 소스 포함 |
| 50-69 | 최근 1년 내 50%+, 최신 소스 일부 |
| 30-49 | 대부분 1년 이상 전 |
| 0-29 | 오래된 정보 위주 |

### 5. Actionability (실행가능성) — 가중치 10% (↓)
트렌드 리서치에서는 **의사결정 근거 제공**이 핵심. 구현보다는 "무엇을 해야 하는가"에 초점.

### 6. Organization (구조) — 가중치 5%
범용 rubric과 동일하되, **트렌드별 정리** 또는 **타임라인 형태**를 선호합니다.

### 7. References (참조 정확성) — 가중치 10%
범용 rubric과 동일.

## SEA 충족률 기준
- Trend rubric에서는 **75% 이상**이 충분 기준 (트렌드는 빠르게 변하므로)
